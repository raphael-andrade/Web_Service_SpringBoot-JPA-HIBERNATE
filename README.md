## Web Service com SpringBoot-JPA-HIBERNATE
### Aplicação com integração ao banco de dados Postgre
 
Nessa api será possivel cadastrar as informações de um cliente para utilizar em algum tipo de comercio, a API será capaz de gerar um pedido de venda com todas as informações necessárias dos produtos escolhidos e no final informando o valor total a ser pago.
Para o desenvolvimento da API foi necessário os seguintes requisitos:
- Lógica de programação e POO
- Conhecimento Avançado em Java e Spring Boot
- Conhecimento em Banco de dados
- Conhecimento em GitHub e Heroku
